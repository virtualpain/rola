Rola UI Framework

Installation

The easiest way to install Rola is simply use bower install rola. It will download all necessary dependencies. Then in your webpage, include these following things into head: 

<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="/css/rola.min.css" media="screen" charset="utf-8">
<link rel="stylesheet" href="/css/rola-other.min.css" media="screen" charset="utf-8">
<script src="/bower_components/jquery/dist/jquery.min.js" charset="utf-8"></script>
<link rel="stylesheet" href="/bower_components/fontawesome/css/font-awesome.min.css" media="screen" title="no title" charset="utf-8">

And these to the near of </body>: 

<script src="/js/rola.min.js" charset="utf-8"></script>
<script src="/bower_components/jquery-modal/jquery.modal.min.js" charset="utf-8"></script>
<script src="/bower_components/tooltipster/jquery.tooltipster.min.js" charset="utf-8"></script>
NOTE You might need to change the URL to the components, and other files.

Grid

Grid system is very easy to use. There are 3 types of container: .container if you want to see different size of container in different breakpoints (mobile/small, tablet/medium, desktop/large). .container-fluid this container is maintain it size in all of our supported breakpoints (90%). And finally .container-full is basically just a wrapper that has 100% width.
